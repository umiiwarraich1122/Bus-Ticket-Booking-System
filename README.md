## 🚌 Bus Ticket Booking System

The Bus Ticket Booking System is a full-stack web application built with React.js, JavaScript, and SQL database. 

This project provides an online platform where users can:
- Register and Login
- Search available buses
- Select seats
- Book tickets
- View booking history

The system uses a relational SQL database to manage user data, bus schedules, seat availability, and booking records efficiently.

This project demonstrates practical implementation of:
- Frontend development using React
- State management and dynamic UI
- Database integration using SQL
- CRUD operations
- Full-stack application structure
